Project Architecture Proposal

Gmind Monorepo & Showcase

Cấu trúc hóa dự án Gmind dưới dạng Monorepo, tách biệt lớp lõi CLI/Agent và đưa các tính năng tinh túy lên Website Showcase (Vercel) để giới thiệu tầm nhìn Agentic Software Engineering 2026.

Part 1: The Showcase Website (Next.js on Vercel)

gmind.gscfin.com

Trang web tĩnh (SSG) với các tương tác thị giác cao cấp. Trình diễn 4 trụ cột 'tinh túy' của nền tảng Gmind.

Pillar A: Code Intelligence

FastCode AST vs RAG

Giải pháp chống "mất trí nhớ cục bộ" của AI. Thay vì RAG vector thuần túy, fastcode map toàn bộ cấu trúc AST (imports, calls) kết hợp BM25 RAG. Trình bày UI so sánh before/after của AI Agent.

Pillar B: SSOT Issue Tracking

FrankenSQLite + Git

Bỏ qua cơ sở dữ liệu cloud, sử dụng DB in-process MVCC lưu trữ local-first. Đồng bộ Git Commit Lineage qua Beads-ID: Trailers. Phô diễn sơ đồ luồng dữ liệu Zero-Dependency.

Pillar C: Verification & SAFe 6.0

Layer 4: Level 3 Gates

Sub-agent Reviewer và CI/CD Gate. Trình diễn Requirements Traceability Matrix (RTM): Trực quan hóa PRD ↔ Plan ↔ Task ↔ Commit trên Kanban Board của PM.

Pillar D: The Ecosystem

Agent Village & Skills

Sự điều phối Agent: 15p Lease Timeout file lock. Thể hiện các workflow như gsafe-research và kỹ năng visual-explainer biến output terminal thành trình duyệt.

Part 2: Monorepo Architecture Layout

Unified Workspace Organization

Chuyển dịch các thành phần repo sang một unified layout, giúp chia sẻ models, gmind protocols và documentation dễ dàng.
